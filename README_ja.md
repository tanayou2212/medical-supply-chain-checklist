# サプライチェーンリスク評価チェックリスト生成ツール

部品カテゴリを入力するだけで、調達リスク評価用のチェックリストをMarkdown形式で出力するツールです。リスクスコアの自動数値計算機能を搭載しています。

---

## 機能

- 部品カテゴリに応じたリスク評価チェックリストを自動生成
- カテゴリ固有の項目 ＋ 共通項目を組み合わせて出力
- **リスクスコア自動計算** — 各項目にy/nで回答すると重み付き総合スコア（0〜100）を算出
- Markdown形式で画面表示 ＋ ファイル保存が可能

### 対応リスク区分

| リスク区分 | 内容 |
|-----------|------|
| 調達先の地域集中リスク | 特定国・地域への依存度 |
| 単一サプライヤー依存リスク | サプライヤー数と財務健全性 |
| 輸送ルートリスク | 代替ルート・関税リスク |
| 代替調達先の有無 | 代替品の認定状況 |
| 在庫・リードタイムリスク | 安全在庫・調達リードタイム |

## 必要環境

- Python 3.8 以上（追加ライブラリのインストール不要）

---

## 使い方

### 方法1：インタラクティブ（対話形式）

```bash
python supply_chain_checklist.py
```

実行後、カテゴリ名を入力し、スコア計算を行うか選択します。

```
==================================================
サプライチェーンリスク評価チェックリスト生成ツール
==================================================

対応カテゴリ例：半導体、センサー、電子部品
  （上記以外のカテゴリも入力可能です）

部品カテゴリを入力してください: 半導体

リスクスコアを自動計算しますか？ (y/n): y
```

`y` を選択すると、全チェック項目について `y`（対応済み）または `n`（未対応）で回答し、スコアが自動計算されます。

### 方法2：コマンドライン引数で指定

```bash
python supply_chain_checklist.py 半導体
python supply_chain_checklist.py センサー
python supply_chain_checklist.py 電子部品
python supply_chain_checklist.py パワーデバイス
```

## リスクスコアの計算方法

各チェック項目への回答：

| 回答 | 意味 | スコアへの影響 |
|------|------|--------------|
| `y` | 対応済み | なし |
| `n` | 未対応 | スコアが上昇 |

**エリアスコア** ＝（未対応項目数 ÷ 全項目数）× 100

**総合スコア** ＝ 各エリアスコアの重み付き平均

| リスク区分 | 重み |
|-----------|------|
| 調達先の地域集中リスク | 25% |
| 単一サプライヤー依存リスク | 25% |
| 代替調達先の有無 | 20% |
| 輸送ルートリスク | 15% |
| 在庫・リードタイムリスク | 15% |

**スコアの判定基準：**

| スコア | リスクレベル |
|--------|------------|
| 0 〜 30 | 低 |
| 31 〜 60 | 中 |
| 61 〜 100 | 高 |

**総合リスクスコア：42.3 / 100　（中リスク）**

| 調達先の地域集中リスク | 50.0 | 中 | 4 / 8 |
| 単一サプライヤー依存リスク | 33.3 | 中 | 2 / 6 |
| 輸送ルートリスク | 50.0 | 中 | 3 / 6 |
| 代替調達先の有無 | 25.0 | 低 | 2 / 8 |
| 在庫・リードタイムリスク | 40.0 | 中 | 4 / 10 |

---

## 調達先の地域集中リスク

- [ ] 台湾・韓国・日本への製造集中度を確認したか
- [ ] 地政学的リスク（台湾海峡情勢）が調達に与える影響を評価したか

## カスタマイズ方法

スクリプト内の以下の辞書を編集することで、チェック項目や重みを追加・変更できます。

- **`CATEGORY_SPECIFIC_ITEMS`**：カテゴリ別の固有チェック項目
- **`COMMON_ITEMS`**：全カテゴリ共通のチェック項目
- **`CATEGORY_ALIASES`**：カテゴリ名の別名定義
- **`RISK_WEIGHTS`**：各リスクエリアの重み（合計が1.0になるよう設定）

例：「バッテリー」カテゴリを追加する場合は、`CATEGORY_SPECIFIC_ITEMS` に以下を追加します。

```python
"バッテリー": {
    "調達先の地域集中リスク": [
        "中国のリチウム・コバルト生産への依存度を確認したか",
        ...
    ],
    ...
}
```
